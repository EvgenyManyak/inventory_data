## 🗺 Roadmap: Inventory Analytics Pipeline (Hybrid Edition)

> **Концепция:** Гибридный пайплайн, сочетающий ручную выгрузку из 1С с автоматизированной обработкой, аналитикой и доставкой отчетов через Telegram.

**🔄 Архитектура потока данных:**
`1С (Excel)` ➔ `/inbox` ➔ `PostgreSQL` ➔ `SQL Analytics` ➔ `PDF/Excel` ➔ `Telegram Bot`

---

### 🐳 Этап 1: Инфраструктура в Docker
- [ ] **Стек:** PostgreSQL, Python Container, Docker Volumes.
- [ ] **Задача:** Поднять базу данных и подготовить структуру общих папок через volumes:
  - `/inbox` — для сырых файлов из 1С.
  - `/processed` — для сведенных файлов.
  - `/out` — для готовых PDF-отчетов.
- [ ] **Результат:** Все инструменты изолированы в контейнерах и готовы к работе.

### 📥 Этап 2: Python Ingestion (Импорт из Excel в БД)
- [ ] **Стек:** Pandas, SQLAlchemy.
- [ ] **Задача:** Написать скрипт-загрузчик (ETL).
  1. Ручное скачивание файлов из 1С и перемещение в `/inbox`.
  2. Python считывает Excel-файлы, очищает и нормализует их в БД (таблица `raw_1c_data`).
- [ ] **Результат:** Все данные из "кривых" Excel-выгрузок лежат в структурированном виде в SQL.

### 📊 Этап 3: Аналитика и Сведение (SQL + Manual)
- [ ] **Стек:** PostgreSQL, DBeaver / psql, Python (для экспорта).
- [ ] **Задача:** Реализовать логику сведения с учетом бизнес-нюансов компании.
  1. Подключение к БД через Python для выполнения сложных SQL-запросов.
  2. Выгрузка результата сведения обратно в Excel для финальной ручной проверки.
- [ ] **Результат:** Чистый, сведенный файл, подтвержденный SQL-расчетами.

### 📄 Этап 4: Генератор PDF-отчетов (Python Reporting)
- [ ] **Стек:** Matplotlib (диаграммы), FPDF2 (верстка PDF).
- [ ] **Задача:** Автоматизировать визуализацию. Python берет сведенный результат из БД и формирует PDF-отчет, включающий:
  - Мини-таблицы по Торговым Точкам (ТТ).
  - Текстовые выводы.
  - Графики потерь.
- [ ] **Результат:** Профессиональный PDF-отчет, готовый к отправке руководству.

### 🤖 Этап 5: Автоматизация рассылки и Архивация
- [ ] **Стек:** aiogram (Telegram Bot), shutil (архивация).
- [ ] **Задача:** Обеспечить доставку и поддерживать порядок в файловой системе.
  1. Telegram-бот отправляет PDF и сведенный Excel руководителю.
  2. Python переносит отработанные исходники в архив и очищает рабочие папки (`/inbox`, `/processed`).
- [ ] **Результат:** Полный бизнес-цикл закрыт и автоматизирован.
